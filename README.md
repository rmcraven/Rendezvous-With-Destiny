《与命运相会》（Rendezvous With Destiny）是“冷战系统——北大西洋地区超时空联动峰会”（以下简称“冷战系统”，前称“北大西洋公约组织超时空联动峰会”）世界观下各类文艺作品构成的半独立世界观。该世界观由“冷战系统”历代主席团成员共同创立并发展。在这里您可以找到该世界观下的正式文章、同人小说和其他文艺作品，以及关于“冷战系统”的基本信息。
